# Smart Bookmark App

Google OAuth based private bookmark manager with realtime updates.

## Features
- Google Login (Supabase Auth)
- Private bookmarks per user
- Realtime sync across tabs
- Tailwind CSS UI
- Deployed on Vercel

## Setup
1. Create Supabase project
2. Enable Google OAuth
3. Create bookmarks table with RLS
4. Add env vars
5. Deploy to Vercel
